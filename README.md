# smrtdesk-web

Static site backing [smrtdesk.online](https://smrtdesk.online).

- `/` — landing.
- `/privacy/` — privacy policy (also mirrored in the macOS app's repo as `PRIVACY.md`).
- `/support/` — support contact + common fixes.

Hosted on GitHub Pages from `main`. CNAME points apex `smrtdesk.online` (and `www.smrtdesk.online`) at `ringo380.github.io`.
